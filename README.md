# Spenza Webhook Frontend

A React.js frontend application for managing webhook subscriptions and monitoring webhook events in real time.

The frontend allows users to:

- Sign up and log in
- Manage webhook subscriptions
- View incoming webhook events
- Monitor event logs in real time

---

# Tech Stack

- React.js
- Axios
- React Router
- JWT Authentication
- CSS / Tailwind

---

# Features

## Authentication
- User Signup
- User Login
- JWT token storage
- Protected routes

## Webhook Management
- Subscribe to new webhooks
- View all subscribed webhooks
- Cancel webhook subscriptions

## Event Monitoring
- View incoming webhook events
- Real-time event updates
- Event log dashboard
